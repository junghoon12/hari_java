<!-- 
        meta 태그
        - HTML 문서(웹 페이지)의 제작자, 내용, 키워드 같은
          여러 정보를 검색엔진이나 웹 브라우저에게 제공.
          - name : 정보의 종류.
          - content : 정보의 값(내용)
    -->
    <!-- 
        문자 인코딩 방식
        - charset(Character Set 단어의 약자)은 문자 인코딩 방식을
          지정하는 HTML 속성임. 문자 인코딩(encoding)이란 문자나
          기호들을 컴퓨터가 알아들을 수 있는 신호로 만드는 것을 말함.
          대표적으로 한글 사용에서는 EUC-KR 또는 UTF-8 등이 사용되며,
          웹에서는 세계 표준 인코딩 방식인 UTF-8 사용을 권장함. 
    -->
    <meta charset="UTF-8">
