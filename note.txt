/*
      String 객체 : 문자열을 다루는 객체.

      - charAt(index) : 문자열에서 인덱스 번호에 해당하는 문자를 반환하는 함수.
      - indexOf("찾을문자") : 문자열에서 왼쪽부터 찾을 문자와 일치하는 문자를 찾아 
                   가정 먼저 일치하는 문자의 인덱스 번호를 반환함. 
                   만일 일치하는 문자가 없으면 -1을 반환함.
      - lastIndexOf("찾을문자") : 문자열에서 오른쪽부터 찾을 문자와 일치하는 문자를 찾아
                   가장 먼저 일치하는 문자의 인덱스 번호를 반환함.
                   만일 찾는 문자가 없으면 null을 반환함.
      - match("찾을문자") : 문자열에서 왼쪽부터 찾을 문자와 일치하는 문자를 찾아 
                   가장 먼저 찾은 문자를 반환함. 
                   만일 일치하는 문자가 없으면 null을 반환함. 
      - reaplce("바꿀문자", "새 문자") : 문자열에서 왼쪽부터 바꿀 문자와 일치하는 문자를
                       찾아 가장 먼저 찾은 문자를 새문자로 변경함.
      - search("찾을문자") : 문자열에서 왼쪽부터 찾을 문자와 일치하는 문자를 찾아
                가장 먼저 일치하는 인덱스 번호를 반환함.
      - slice(a, b) : a 개의 문자를 자르고 b번째 이후에 문자를 자른 후,
                   남은 문자를 반환함.
      - substring(a, b) : a인덱스부터 b인덱스 전까지 구간의 문자를 반환함.
      - substr(a, 문자갯수) : 문자열에 a인덱스부터 지정한 문자 갯수만큼 문자열을 반환함.
      - split("문자") : 지정한 문자를 기준으로 문자를 나누어 배열에 저장하여 반환함.
      - toLowerCase() : 문자열에서 영문 대문자를 모두 소문자로 바꾸어줌.
      - toUpperCase() : 문자열에서 영문 소문자를 모두 대문자로 바꾸어줌.
      - length : 문자열에서 문자의 갯수를 반환해 줌.
      - concat("새로운 문자") : 문자열에 새로운 문자열을 결합함.
      - charCodeAt(index) : 문자열 index에 해당하는 문자의 아스키 코드 값을 반환함.
      - fromCharCode(아스키 코드값) : 아스키 코드값에 해당하는 문자를 반환함.
      - trim() : 문자의 맨 앞과 맨 뒤에 있는 공백 문자열을 제거함.    
    */
