# requests 모듈은 HTTP 요청을 보내는데 사용이 되는 모듈임.

# Beautiful 모듈은 HTML이나 XML 문서를 파싱하고 분석하는데 사용되는 모듈임.

#     웹 페이지의 특정 부분을 추출하거나 원하는 데이터를 수집하는데 사용이 됨.
