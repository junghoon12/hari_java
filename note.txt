* 웹 표준(Web Standard)?
  - 월드 와이드 웹(www)을 서술하고 정의하는 공식 표준이나
    기술 규격을 가리키는 일반적인 용어.
  - 최근에 이 용어는 웹 사이트를 작성하는데 중요도가 높아지고
    있으며, 웹 디자인, 개발과 관계가 있음.
  - 웹 사이트 개발 시에는 HTML, XML, CSS, JavaSxript 
    등 여러 가지 언어를 사용하게 되는데, 이 때 미리 정의된
    규칙과 규격에 맞추어 만든다면 그것이 웹 표준이라고 생각하면 됨.
  - ISO에서 제정한 품질경영시스템에 관한 국제규격 ISO9001 
    이란 걸 들어 보았을 것임.
  - 웹 표준이라는 것도 결국은 그와 같은 "웹 페이지 제작 도구와
    기술에 대한 규격" 이라고 생각하면 됨.
  - 웹 표준을 정의한 단체 : W3C(World Wide Web Consortium)

  
* 웹 표준이 중요한 이유??
  - 일단은 표준을 잘 지킨다면 관리적인 측면에서 유리한 점이 많음.
  - 체계적인 가이드 라인에 따라 만들어 졌다면 웹 개발자 누구라도
    이해하기가 쉬워짐. 하지만, 자신만이 알 수 있는 기술방식이라면
    전문가라도 해당 웹 페이지를 이해하는데 어려움이 있을 것임.
    즉, 알기 힘든 웹 페이지는 만든 제작자만이 수정해야 하는
    난처한 상황을 만들 수 있음.
