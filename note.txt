<%-- 
	     <fmt:formatDate> 태그의 여러가지 속성.
	     
	     - value : 포맷될 날짜를 지정하는 속성.
	     - type : 포맷팅할 타입을 지정하는 속성.
	              * date : 날짜를 지정.
	              * time : 시간을 지정.
	              * both : 날짜 / 시간을 모두 지정.
	     - dateStyle : 날짜의 출력 양식을 지정하는 속성.
	                   값(value)에는 full, long, medium, short 
	                   등을 지정할 수 있음.
	     - timeStyle : 시간 출력 양식을 지정하는 속성.
	     - pattern : 직접 출력 양식을 지정하는 속성.
	     - timeZone : 특정 나라 시간대로 시간을 설정하는 속성. 
	 --%>
