<!-- 
	    선택 객체의 변수식
	    - 변수식은 기본적으로 컨트롤러 측이 준비한 값을 그대로 출력하는 것이 전부임.
	      하지만 숫자나 텍스트를 값으로 사용할 때는 문제가 없지만 객체를 사용하려고
	      하면 문제가 발생함.
	    - 물론 ${object.name} 처럼 객체의 프로퍼티나 메서드를 지정해서 사용하면
	      되지만 객체 안에 다수의 값이 존재하는 경우 일일이 작성하는 것은 매우 귀찮은
	      일이 됨. 그 뿐만 아니라 객체 이름이 변경이 되면 정의한 이름들도 모두 변경해야
	      하므로 감당하기 힘든 작업이 됨.
	    - 이런 경우 객체를 지정해서 해당 객체 안에 있는 값을 추출할 수 있는 전용 변수식을
	      사용하면 됨.
	      
	      형식) *{  }
	            <OO th:object="${객체키}">
	              <OO th:text="*{프로퍼티}">
	    
	    - 태그에 "th:object" 라는 속성을 이용하여 객체를 설정함.
	      이렇게 하면 해당 태그 내부에서 *{} 을 이용한 변수식을 사용할 수 있게 됨.
	      이 별표(*) 형태의 변수식에서는 객체 내의 프로퍼티 등을 이름만으로 지정 가능.
	 -->
