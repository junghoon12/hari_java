/*
 * java에서의 입출력 방식
 * - 스트림이라는 방식을 이용함.

 *                    즉, 데이터가 물 흐르는 것처럼
 *                    흐른다고 생각하면 됨.

 * - 스트림의 종류

 *   2) 문자 스트림 : 2바이트 단위로 데이터를 입출력(char).
 * - java 입출력 관련 클래스 : java.io 패키지에 존재함.
 *        ==> 해당 패키지에 있는 클래스들을 이용하여 파일을 
 *            입출력함.
 * - checked 방식의 예외 처리가 적용됨.
 * - java의 입출력은 단방향성 ==> 한쪽으로만 입력되고 출력이 됨.
 * - FIFO(First In First Out) : 먼저 입력되고 먼저 출력이 됨.  
 */
